# go 语言实现简单聊天室

------

从头开始，每日精进，实现一个比较完整的聊天室，简单的 mvc 架构 具有以下功能点:

> * 私聊，群聊
> * 发送文字，表情包，图片，语音，视频
> * websocket的使用，golang 并发优化
> * 分布式部署方案
